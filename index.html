How it works

reepatts is a focused static analyzer. It does one thing and tries to do it well.

  1. Fetch bytecode from the Pharos RPC via eth_getCode.
  2. Scan the bytecode with a PUSH-data-aware opcode walker.
  3. Match against 6 canonical reentrancy fingerprints.
  4. Score each finding 0–100 (HIGH, CRITICAL, etc.).
  5. Annotate with the 4-byte function selector the finding lives in.

Severity is the base score; the matcher adds a +15 cross-function bonus, a −10 ReentrancyGuard penalty (when the contract pushes to slots 0x4f10 / 0x6d10 / 0x3659), and other modifiers. Final score is clamped to [0, 100].

Source & install

The full skill lives at github.com/pazzy422/reepatts. Install it as a Pharos Agent Center skill, or use the Python / bash forms directly:

git clone https://github.com/pazzy422/reepatts.git
cd reepatts
bash scripts/scan.sh 0xYOUR_CONTRACT --network mainnet

11/11 pattern-matcher unit tests pass. Read-only — no private key required.
