# reepatts-demo

Live browser demo of [reepatts](https://github.com/pazzy422/reepatts) — a static analyzer for reentrancy patterns in deployed EVM bytecode, built for the Pharos Network.

**Live site:** https://pazzy422.github.io/reepatts-demo/

## What this repo is

A single static `index.html` that runs the same pattern-matcher logic as the reepatts Python/bash scripts — but in the browser. It calls the live Pharos RPC via CORS and renders the report in a chat-style agent conversation UI.

## Source

The skill itself lives at https://github.com/pazzy422/reepatts.
